# Contributing to express-err-helper

Thanks for your interest in contributing to **express-err-helper**! 🎉
We welcome **bug fixes, improvements, documentation updates, and new features**.

---

# Quick Contribution Guide

### 1. Fork the repository

Click **Fork** on GitHub to create your own copy.

---

### 2. Clone your fork

```bash
git clone https://github.com/<your-username>/express-err-helper.git
cd express-err-helper
```

---

### 3. Create a new branch

```bash
git checkout -b my-change
```

Example branch names:

```
fix/error-handler-bug
feat/add-status-code
docs/update-readme
```

---

### 4. Make your changes

Edit the code or documentation.

Before committing, run formatting:

```bash
npm run format
```

---

### 5. Commit your changes

```bash
git add .
git commit -m "fix: improve error handler"
```

---

### 6. Push your branch

```bash
git push origin my-change
```

---

### 7. Open a Pull Request

Go to the GitHub repository and click **New Pull Request**.

Explain:

* what you changed
* why the change is needed

---

# Reporting Issues

Found a bug or have an idea?

Open an issue here:

https://github.com/mohdanas86/express-err-helper/issues

Please include:

* Node.js version
* Express version
* A small example of the problem

---

# Code Guidelines

* Keep code **simple and readable**
* Follow existing **project structure**
* Avoid adding unnecessary dependencies
* Run `npm run format` before committing
