# Security Policy

## Supported Versions

We actively maintain the latest version of **express-err-helper**.

If you discover a security vulnerability, please report it responsibly.

---

## Reporting a Vulnerability

Please **do not open a public issue for security vulnerabilities**.

Instead, report the issue privately to the project maintainer.

You can contact the maintainer through GitHub:

https://github.com/mohdanas86

Include the following information:

* Description of the vulnerability
* Steps to reproduce the issue
* Possible impact
* Suggested fix (if available)

---

## Response Policy

Once a vulnerability report is received:

1. The issue will be reviewed as soon as possible.
2. A fix will be developed if necessary.
3. A new secure version will be released.

We appreciate responsible disclosure to keep the project safe for everyone.
